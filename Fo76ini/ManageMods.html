ABOUT ARCHIVE2

In order to use the mod manager, you need to install Archive2.
Archive2 is the official tool to pack/extract *.ba2 archives.
It is shipped with Fallout 4's Creation Kit.
Note that Skyrim's Creation Kit won't cut it, you need Fallout 4.

I'm not going to distribute Archive2 with my tool because I don't want any legal issues. If you choose to pirate it, that's your pigeon. Also, I think most people who play and mod Fallout 76 already own Fallout 4 and have the Bethesda.net Launcher installed.

How to install Archive2:

  1. Install Fallout 4.
  2. Get the Bethesda.net Launcher, create a account if needed.
  3. Install "Creation Kit: Fallout 4" (see Figure 1)
  4. Open the "Manage mods" window and set the path to "Archive2.exe". (see Figure 2)
    You can find Archive2.exe in <your steam folder>\steamapps\common\Fallout 4\Tools\Archive2\Archive2.exe.


FAQ

How to add mods?
Click on File > Add mod > From archive.
You can also drag and drop them into the mod order.

How to completely disable mods?
Check "Nuclear Winter Mode" and click on "Deploy". It disables mods entirely.
The sResourceIndexFileList will temporarly be saved in QuickConfiguration.ini instead of the Fallout76Custom.ini, so you don't lose your manually installed mods.

What happens to mods I've installed manually?
The tool won't touch these mods. If you want to manage them, you can import them: File > Import installed mods

Can I get banned for using mods?
Yes, of course. In fact, you can get permanently banned.
Also, refer to Bethesda's support page: https://help.bethesda.net/app/answers/detail/a_id/44429/kw/mod

Why don't you distribute Archive2 with your tool?
Because Bethesda owns the Archive2 tool and if I did so, I'd basically pirate it.
Other mod authors might do this, but I don't want any legal issues.
